# Propuesta de Mejora

Si se contara con más tiempo de desarrollo, se podrían implementar las siguientes mejoras:

## 1. Paginación de personajes
Actualmente la aplicación muestra únicamente la primera página de resultados de la API.

Como mejora, se podría implementar un sistema de paginación que permita navegar entre las distintas páginas de personajes, mejorando la experiencia de usuario y el acceso a un mayor volumen de información.

## 2. Integración de episodios por personaje
Cada personaje de la API cuenta con información sobre los episodios en los que aparece.

Una mejora sería mostrar una lista de episodios asociados a cada personaje, ya sea al expandir la tarjeta o mediante una ventana emergente, brindando mayor contexto y profundidad a la información presentada.
